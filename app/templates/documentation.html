Modèle LSTM

Le modèle LSTM (Long Short-Term Memory) est un type de réseau de neurones récurrents particulièrement adapté à l'analyse de séries temporelles.

Architecture :

  • Couche d'entrée : 4 features (débit, pression, demande, niveau)
  • 2 couches LSTM avec 128 et 64 unités

Format des Données

Les données d'entrée doivent respecter le format CSV standard.
